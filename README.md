## gym/

Practice problems grouped by CodeForces rating:
- r800
- r900
- r1000
- r1300

Each problem usually includes:
- a C++ solution
- a Python solution

File naming format:
<problemID>_<problem_name>.<extension>

Example:
263A_Beautiful_Matrix.cpp  
263A_Beautiful_Matrix.py
